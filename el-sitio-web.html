--- title: El sitio web ---

Como funciona el sitio web?

Este sitio es solo front-end, es decir todo lo que vez es lo que hay, no tenemos bases de datos, ni usuarios, nada (Revisa este artículo para mas informacion). Entonces, como es que haces funcionar esto? El sitio web es HTML, CSS y JavaScript. HTML y CSS se encargan de mostrar el texto y darle el estilo respectivamente, permitiendo una mejor experiencia visual. JavaScript se encarga de darle funcionamiento al sitio web, por ejemplo: - La busqueda - Las herramientas - Mostrar algunos datos - Etc.

Mis datos se guardan?

Nosotros respetamos tu privacidad, creemos que es un derecho fundamental, por lo que no, no almacenamos ningun dato que nos indique directamente quien eres, como nombre, correo, direccion fisica, numero de telefono, documento de identidad, etc. Tal vez servicios externos como Cloudflare, Arc.io, Algolia y otros puedan recopilar datos como un identificador unico, etc. Pero esto no afectará tu visita.
